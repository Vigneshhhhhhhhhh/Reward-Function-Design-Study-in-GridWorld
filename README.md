# Reward Function Design Study in GridWorld  
From stochastic pathfinding to reward engineering — a comprehensive study on agent convergence and policy optimality.

- **MDP Environment**: A 5x5 to 20x20 stochastic grid with slip probabilities.
- **Dynamic Programming**: Optimized Value Iteration and Policy Iteration solvers.
- **Sparse vs Dense**: Comparative analysis of end-goal rewards vs state-shaping signals.
- **Negative Step Penalty**: Automated shortest-path optimization via time-decay penalties.
- **Deceptive Incentives**: Study on how local maxima 'trick' agents away from global goals.
- **Automated Visualization**: High-DPI heatmaps, policy arrows, and convergence GIFs.
- **Presentation Ready**: Dedicated Jupyter and Colab notebooks for live demonstrations.
- **Statistical Rigor**: Wilcoxon Signed-Rank tests for convergence significance.
